# pytorch-attention-mechanism
my codes for learning attention mechanism

## CNN with attention

### Usage

Train the model:

```bash
$ python cnn-with-attention.py --train
```

Visualize attention map:

```bash
$ python cnn-with-attention.py --visualize
```
